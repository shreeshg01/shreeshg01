## 🏆 Projects

### 🚶 Reinforcement Learning for Human-Like Locomotion
Trained a PPO agent to learn stable, efficient walking from scratch in a MuJoCo physics simulator — no reference motion data — and studied whether the learned gait shows human-like locomotion features.
- End-to-end RL pipeline using Gymnasium, Stable-Baselines3, and MuJoCo
- Multi-component reward design (velocity, posture, energy, fall penalty) to shape natural gait
- Iterative training and gait analysis across Walker2d and Humanoid environments

👉 Repository: https://github.com/shreeshg01/Reinforcement_Learning_For_Humanlike_Locomotion_using_MuJuCo

### 🗺️ Spatial Accident Clustering & OLAP Analysis
Density-based clustering and multi-dimensional OLAP analysis of 500K+ US traffic accidents to surface and validate high-risk hotspots.
- DBSCAN (Haversine metric) with automated `eps`/`min_samples` tuning to detect geographic hotspots
- Cluster profiling, Silhouette + Davies-Bouldin validation, and grid/K-Means baseline comparison
- Multi-dimensional OLAP cubes and an interactive Folium hotspot map

👉 Repository: https://github.com/shreeshg01/Spatial_Accident_Analysis

### 🧘 Yoga Pose Estimation and Prediction
Deep learning–based system for predicting yoga poses from images and videos using pose estimation and CNNs, with an emphasis on real-time feedback.
- Pose detection and classification pipeline
- Real-time inference on images and videos
- Practical application of computer vision concepts

👉 Repository: https://github.com/shreeshg01/Yoga_Pose_Estimation_and_Prediction
